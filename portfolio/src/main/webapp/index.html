Projects

In the summer of 2019, I participated in the CSSI program at Google. While there I learned the basics of how to develop the front end and back end of a website using CSS, Javascript, HTML, and Python and hosting it on App Engine. The website my team and I created was a social media-esque website where users could make and sign into accounts, and from those accounts they could access their saved posts and comments by recording the likes they made with datastore.

My role in the project was to implement the datastore that would save user data, the posts they would like, and to implement the like functionality that would prevent the user from liking more than once. I also implemented the function and page that would allow the user to actually post their meme to the website.

The name of our website is called MeTwo!!!
